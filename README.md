# Mesh X3 Viewer

使用自定義 Mesh 數據並使用 Threejs 繪製的 Viewer

首先它並非載入一般大家熟系的制式模型格，而是使用自定義的數據(不外乎是一些頂點資訊與群組分類)，所以不是我的 Viewer 是無法解析這個模型的。數據載入後使用 Threejs 直接繪製出來

實際上它只是近期為了面試機會很快的碼一個小項目給人看...
